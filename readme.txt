== Installation ==
 
Here's how to install the Snap Ecommerce plugin.
 
1. Download the plugin and unzip it's contents into a folder. Upload the contents of this folder to your plugins directory (normally /wp-content/plugins/)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the setup instructions and start selling!
